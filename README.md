## Overview

### Quick start

```sh
git clone https://github.com/GeorgStrassberger/vue_shop.git
```

```sh
cd ./vue_shop
```

```sh
npm install
```

```sh
cp example.env .env
```

```sh
npm run dev
```

or

```sh
npm run build
```

```sh
npm run test:unit -- --run
```
